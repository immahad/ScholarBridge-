ScholarBridge Admin Login Instructions

This document provides instructions for logging into the ScholarBridge admin portal.

Admin Credentials

Email: admin@example.com

Password: Admin@123

Admin URL: http://localhost:5173/login

Login Steps

  1. Navigate to http://localhost:5173/login
  2. When you arrive at the login page, click on the "Admin Login" button at the bottom
  3. The form will switch to admin login mode (you'll see "Admin Login" as the header)
  4. Enter the admin email: admin@example.com
  5. Enter the admin password: Admin@123
  6. Click the "Login as Admin" button
  7. You will be redirected to the admin dashboard if login is successful

Note

If you encounter any issues logging in:

  • Make sure the backend server is running at http://localhost:5000
  • Make sure the frontend server is running at http://localhost:5173
  • Check browser console for any error messages
  • If account is locked, run the unlock script: node scripts/unlockAdmin.js
